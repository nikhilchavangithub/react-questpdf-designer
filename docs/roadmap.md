# Roadmap

## Phase 1

- Three-panel React designer
- Absolute Letter canvas
- Text, box, line, image placeholder, and table placeholder
- Zustand state and immutable schema updates
- JSON preview and PDF generation
- ASP.NET Core `/api/pdf/generate` endpoint
- QuestPDF absolute rendering with simple bindings

## Phase 2

Add flow layout slots, native rows/columns/tables, data-bound repeaters, and page-aware layout.

## Phase 3

Add template persistence/versioning, reusable components, conditional visibility, expression rules, page breaks, CMS1500 overlays, print calibration, and PDF diff testing.
